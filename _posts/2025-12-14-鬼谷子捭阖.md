~~~
“鬼谷子”原名王诩，春秋战国时期卫国朝歌(今河南省鹤壁市淇县)人，是纵横家鼻祖。
春秋战国时期著名的道家、兵家、思想家、谋略家、教育家，通天彻地，兼顾数家学问。
他是中国历史上一位极具神秘色彩的人物，被誉为千古奇人，长于持身养性，精于心理揣摩，深明刚柔之势，通晓纵横捭阖之术，独具通天之智，是兵家著名代表。
他的弟子有兵家：孙膑、庞涓，纵横家：张仪、苏秦。
据说鬼谷子常入云梦山采药修道。因隐居周阳城清溪之鬼谷，故自称鬼谷先生。
~~~


# 引言

“捭阖”是鬼谷子的开篇，`捭`即开启，`阖`即闭藏。
`捭阖`包含阴阳、进退、开闭、柔刚、大小、高低、贵贱等多宗含义，也就是“辩证”的哲学概念。
既是万事万物的发展变化规律，也是纵横家如何进行游说的根本方法。

所谓的“捭阖之术”就是开合有道、张弛有度

即：`何时应敞开心扉，直言陈辞；何时应冷静观察，沉默不语`。通过捭与阖的相互配合，不难洞悉对方的虚实真相，从而达到知人、御人的目的。

# 正文

<br>

```tex
📕 % 原文
粤若稽古，圣人之在天地间也，为众生之先。观阴阳之开阖以命物，知存亡之门户，筹策万类之终始，达人心之理，见变化之眹焉，而守司其门户。故圣人之在天下也，自古及今，其道一也。

% 译文
纵观古今历史，可知圣人生活在世界上，就是要成为众人的先导。通过观察阴阳两类现象的变化来对事物作出判断（通过观察事物的对立统一变化，来辨别事物的发展），并进一步了解事物生存和死亡的途径。计算和预测事物的发展过程，通晓人们思想变化的规律，揭示事物变化的征兆，从而把握事物发展变化的关键。所以，从古到今，圣人所遵循的方法、守则始终一致。

📕 % 原文
变化无穷，各有所归。或阴或阳，或柔或刚，或开或闭，或弛或张。是故圣人一守司其门户，审察其所先后，度权量能，校其伎巧短长。

% 译文
世间事物的变化是无穷无尽的，然而都各有自己的归宿：或者属阴，或者归阳；或者柔弱，或者刚强；或者开放，或者封闭；或者松驰，或者紧张。
所以，圣人始终可以把握事物发展变化的关键，审察事物的前因后果、轻重缓急。权衡对方的思维能力和实践能力，比较其技巧优劣长短，然后借物举事。

```


🤔几点思考：
* 圣人之所以为圣人，根本就是顺应时代发展的潮流，遵循天下兴亡之道。
* 万事万物的规律是类似的，因此，圣人或者说出色的人，无论从事哪一方面，都要有从事物微小的迹象中得到事物走向和发展趋势的能力。
* 日常的一些现象，人人都习以为常，但它们的细小变化中，往往存在一些道理和启示，圣人们却有这样的敏锐，从习以为常的现象（或微小的事物）中得到哲理/规律，看到事物本质。


<br>

```tex
📕 % 原文
夫贤不肖、智愚、勇怯有差，乃可捭，乃可阖；乃可进，乃可退；乃可贱，乃可贵，无为以牧之。

% 译文
人们的资质是有差别的，有的贤能，有的不肖；有的聪明，有的愚蠢；有的勇敢，有的怯懦。
针对不同的人，要用不同的对待方式，可以对人开放，也可以对人封闭；可以引进（提升），也可以辞退；可以轻视，也可以尊重，要用无为的方式掌握他们(顺乎自然地加以驾驭)。

📕 % 原文
审定有无与其实虚，随其嗜欲以见其志意。微排其所言而捭反之，以求其实，实得其指；阖而捭之，以求其利。

% 译文
详查对方有什么、缺什么，以了解其虚实；通过分析他的嗜和欲望，以摸清其意愿。适当贬抑对方所说的话，当他们开放以后再反复考察，以便探察实情，切实把握对方言行的宗旨；
通过让对方先封闭而后开放（采纳和质疑对方的言论，使对方一步步袒露心扉），以便获得自己想要的利益。

📕 % 原文
或开而示之，或阖而闭之。开而示之者，同其情也；阖而闭之者，异其诚也。可与不可，审明其计谋，以原其同异。离合有守，先从其志。

% 译文
或者把实情向对方公开，或者把秘密隐藏。与对方感情一致，则敞开心扉；与对方心意不同，则关闭心扉。
敞开心扉与否，取决于是否研究清楚对方的计谋/想法，明白自己和对方在本质问题上的差异。不管是离是合，都必须要等待时机，尽量先满足对方意愿，然后适时而动。

```

🤔几点思考：
* 现实生活中，不同的人，应采取不同的对策，根据别人不同的特性，采取完全不同的对待方法。
* 要学会见什么人就说什么话，而要做到这点就需要全面地了解别人，也只有全面而深刻地了解别人，才能更好的“求其利”。
* 要想说服别人，就应该先站于别人的角度，以“为你好”、“为了着想”等别人的立场出发，达到自己的目的。

```tex
📕 % 原文
即欲捭之贵周，即欲阖之贵密。周密之贵微，而与道相追。

% 译文
如果要运用开启之术，务必要谋划周全（没有遗漏）；假如要运用闭合之术，贵在隐藏和保密。由此可见，周详和保密的重要，几乎与“道”相通。

📕 % 原文
捭之者，料其情也；阖之者，结其诚也。皆见其权衡轻重，乃为之度数，圣人因而为之虑。其不中权衡度数，圣人因而自为之虑。

% 译文
之所以要让对方开启，是为了判断对方的实情；让对方封闭，是为了坚定他的诚心。这样做的目的，都是为了能使对方显露实情（摸清对方底细），以权衡比较谋略的得失程度，圣人也是按照这样的方法进行考虑。假如不能测出对方的程度和数量，圣人为此会自我忧虑（也会想办法另行谋划）。

📕 % 原文
故捭者，或捭而出之，或捭而纳之；阖者，或阖而取之，或阖而去之。捭阖者，天地之道。捭阖者，以变动阴阳，四时开闭，以化万物。纵横反出，反覆反忤，必由此矣。

% 译文
因此，所谓开启，有的是要别人自己将真实情况暴露出来，有点事让别人接纳自己的观点；
所谓封闭，或者是闭藏之后而获取，或者闭藏之后而放弃。
开启和闭合，是天地之间运行的规律。开启和闭合，也是阴气和阳气的变化运动，四时节令的开始和终止变化也如同开启和闭合一样，是为了促进事物的发展变化。不论纵横与反复都必须经过开启和闭藏来实现。

```

🤔几点思考：
* 捭阖之术，归根结底就是要琢磨对方，然后想出对策，进而到达趋利避害的目的。对自己有利就要开放，对自己不利就要封闭。
* “捭阖”原本是指门对开关，人的心中也需要有这样的一扇门，知道应该把什么关在门内，什么放在门外。
* 捭阖之术要确保周详缜密，攻守兼备。在“闭”的时候能确保自守门户，韬光养晦，渡过难关，从而占据先机，一役而胜。
