~~~
“鬼谷子”原名王诩，春秋战国时期卫国朝歌(今河南省鹤壁市淇县)人，是纵横家鼻祖。
春秋战国时期著名的道家、兵家、思想家、谋略家、教育家，通天彻地，兼顾数家学问。
他是中国历史上一位极具神秘色彩的人物，被誉为千古奇人，长于持身养性，精于心理揣摩，深明刚柔之势，通晓纵横捭阖之术，独具通天之智，是兵家著名代表。
他的弟子有兵家：孙膑、庞涓，纵横家：张仪、苏秦。
据说鬼谷子常入云梦山采药修道。因隐居周阳城清溪之鬼谷，故自称鬼谷先生。
~~~


# 引言

「抵巇」篇把「缝隙」当作一切危机与机会的共同源头：`巇`是裂隙，`抵`是顶住、填塞、撑开、利用或顺势推翻的一整套手法。
物有自然之理，事有离合之势；裂隙刚露时细如秋毫，拖下去却能动摇根本。本篇既讲怎样「塞」与「息」——修补、止损、维稳；也讲在不可救药时怎样「得」——取而代之、重建秩序；最后把圣人定位为能察隙、能待时、能上合下检的「天地之使」。读这篇不必只想到庙堂权变：组织裂痕、舆情危机、合作破裂，都是当代的「巇」。

# 正文

<br>

```tex
📕 % 原文
物有自然，事有合离。有近而不可见，有远而可知。近而不可见者，不察其辞也；远而可知者，反往以验来也。

% 译文（白话）
万物各有自己的规律，事情也总在聚合与分离之间来回。于是会出现怪现象：近在眼前却看不清，远在千里反而能料中。近在眼前却看不清，往往是因为没有细察对方的言辞与实情；远却能知，多半是因为懂得用往事、用往返验证来推断将要发生的事。
（这一段与《反应》篇「反往验来」相呼应：距离不是问题，方法才是问题。）

```

🤔几点思考：
* 很多误判不是信息太少，而是「太近」带来的盲区——听其辞而不察其实。
* 「反往以验来」可作复盘习惯：用旧案例对齐新苗头。

<br>

```tex
📕 % 原文
巇者，罅也。罅者，涧也，涧者成大隙也。巇始有朕，可抵而塞，可抵而却，可抵而息，可抵而匿，可抵而得，此谓抵巇之理也。

% 译文（白话）
所谓「巇」，起初只是细缝（罅）；罅加深就像山涧；涧再扩就成了大裂隙。裂隙刚有征兆时（有朕），可以用「抵」去处理：塞住它、推开它、止住它、让它消匿于无形，或者在局面转换时「抵」出一个新的可据之位——这些都是「抵巇」的道理。
要点是：动手越早，成本越小；等裂到「涧」「大隙」，往往就只能选更激烈的那几手了。

```

🤔几点思考：
* 项目管理里的风险登记册、工程里的细小裂缝，都是「朕」；怕的是视而不见。
* 「塞、却、息、匿、得」五字可理解为工具箱，不是每次都要「得」；能塞则塞，才是上工治未病。

<br>

```tex
📕 % 原文
事之危也，圣人知之，独保其用。因化说事，通达计谋，以识细微。经起秋毫之末，挥之于太山之本。其施外，兆萌芽蘖之谋，皆由抵巇。抵巇之隙，为道术。

% 译文（白话）
事情刚露出危险苗头时，圣人能觉察，并且能独自把可用之力保全下来。顺着变化去说事、通盘去用计，才能在极细处识别吉凶。事情往往从「秋毫之末」那样小的地方起头，却可能撼动像泰山根基那样大的局面。圣人施教于外、防范于初，那些刚发芽的阴谋与祸端，都要靠「抵巇」这套来处置。可以说：在缝隙上做文章，已经上升为一道「道术」，而不只是临时救火。

```

🤔几点思考：
* 「秋毫」与「太山」是整篇的修辞眼：战略来自对微兆的敏感。
* 「独保其用」可理解为：别在恐慌里把牌一次性打光，要留能转圜的力。

<br>

```tex
📕 % 原文
天下分错，上无明主，公侯无道德，则小人谗贼，贤人不用，圣人窜匿，贪利诈伪者作，君臣相惑，土崩瓦解，而相伐射，父子离散，乖乱反目，是谓萌牙巇罅。圣人见萌牙巇罅，则抵之以法。世可以治则抵而塞之，不可治则抵而得之；或抵如此，或抵如彼，或抵反之，或抵覆之。五帝之政，抵而塞之；三王之事，抵而得之。诸侯相抵，不可胜数。当此之时，能抵为右。

% 译文（白话）
天下纷乱，上无明君，公侯失德，于是小人进谗、贤者见弃、圣人避世，贪诈之徒蜂起，君臣互相猜疑，纲纪像土崩瓦解，百姓互相攻伐，父子离散、恩义反目——这就叫「萌牙巇罅」：大乱的裂缝已经冒芽。圣人看见这种芽，就要用「抵」的法度去对付。
若世道还可救，就「抵而塞之」：堵缝、弥合、恢复秩序；若已不可救，就「抵而得之」：在旧结构的裂口上另立新的可运行框架。于是有塞、有得、有反、有覆等不同取径。传说五帝时代多能「塞」；三王起事则往往以「得」成其功；诸侯之间用「抵」相争的，更数不胜数。大乱之世，谁最善于在缝隙处下手，谁就被推为右（上选）。

```

🤔几点思考：
* 这一节政治色彩很重，当作历史修辞读即可；抽掉朝代，剩下的是危机分型：可修补 vs 须换代（或须换架构）。
* 「能抵为右」在组织语境里，常对应：能在矛盾爆发前牵头止损的人，反而比只会事后表态的人更稀缺。

<br>

```tex
📕 % 原文
自天地之合离、终始，必有巇隙，不可不察也。察之以捭阖，能用此道，圣人也。圣人者，天地之使也。世无可抵，则深隐而待时；时有可抵，则为之谋。可以上合，可以检下。能因能循，为天地守神。

% 译文（白话）
自从天地有合有离、有终有始以来，就一定伴随着「巇隙」相随，不能不明察。要用捭阖开合的眼力去察，又能把抵巇之道用活，才称得上圣人。圣人是「天地之使」：在结构里承担察变、弥缝或更化的责任。
世间暂时无缝可抵，就深藏待时；一旦缝隙出现，就为天下（或为主事者）设谋。如此则上能与势相合，下能督察、收束民情与吏治。既能因势利导，又能循理守法，于是就像替天地守住那口气、那尊神——也就是守住秩序的根本。

```

🤔几点思考：
* 「无可抵则隐」不是躺平，而是不在真空里硬造矛盾；与《反应》之静听、《内揵》之待时一脉相承。
* 「能因能循」提醒：权变要有底线，否则「抵」就变成纯破坏。
