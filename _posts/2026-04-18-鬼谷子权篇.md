~~~
“鬼谷子”原名王诩，春秋战国时期卫国朝歌(今河南省鹤壁市淇县)人，是纵横家鼻祖。
春秋战国时期著名的道家、兵家、思想家、谋略家、教育家，通天彻地，兼顾数家学问。
他是中国历史上一位极具神秘色彩的人物，被誉为千古奇人，长于持身养性，精于心理揣摩，深明刚柔之势，通晓纵横捭阖之术，独具通天之智，是兵家著名代表。
他的弟子有兵家：孙膑、庞涓，纵横家：张仪、苏秦。
据说鬼谷子常入云梦山采药修道。因隐居周阳城清溪之鬼谷，故自称鬼谷先生。
~~~


# 引言

`权`是权衡、权变：本篇专论「言辞的品类与功用」——说、饰、应对、成义、难言各是什么路数；佞、谀、平、戚、静五种话术各打什么算盘；再落到口、耳、目与心如何「参调」，以及「九言」（对智、拙、辩、贵、富、贫、贱、勇、过九类人）的取径。末段提出辞言五种病色（病、怨、忧、怒、喜），并归结到「听贵聪，智贵明，辞贵奇」。与《飞箝》《揣》《摩》连读，权篇相当于「输出层规范」：同样的事实，如何因人、因时、因势选择声口。

# 正文

<br>

```tex
📕 % 原文
说者，说之也；说之者，资之也。饰言者，假之也；假之者，益损也。应对者，利辞也；利辞者，轻论也。成义者，明之也；明之者，符验也。难言者，却论也；却论者，钓几也。佞言者，谄而干忠；谀言者，博而干智；平言者，决而干勇；戚言者，权而干信；静言者，反而干胜。先意承欲者，谄也；繁称文辞者，博也；策选进谋者，权也；纵舍不疑者，决也；先分不足而窒非者，反也。

% 译文（白话）
所谓「说」，是去说服对方；能说服对方，也是在给对方提供一种可采纳的「资粮」。所谓「饰言」，多借假于文采与夸饰；既假借，就必有增有减，不尽属实。所谓「应对」，贵在口利；口利往往流于轻论。所谓「成义」，要把道理讲明白；讲明白是为了经得起符验。所谓「难言」，是反方向的驳论；驳论往往意在钓出对方隐藏的几微。
接下来五种声口：佞言以谄求忠名，谀言以浮夸求智名，平言以果决求勇名，戚言以忧形求信名，静言以暗讽求胜名。再五种行径：先意承欲是谄，堆砌文辞是博，精选策谋以进是揽权，取舍不疑是果决，自己分量不足却先堵人之口、责人之非，是反噬一路。

```

🤔几点思考：
* 这一段像「话术类型学」：先能命名，才不容易被话术带着跑。
* 「成义者，符验」与《摩篇》内外相符同一逻辑：能应验的才叫义理，否则只是利辞。

<br>

```tex
📕 % 原文
故口者，机关也，所以关闭情意也。耳目者，心之佐助也，所以窥间奸邪。故曰参调而应，利道而动。故繁言而不乱，翱翔而不迷，变易而不危者，睹要得理。故无目者，不可示以五色；无耳者，不可告以五音。故不可以往者，无所开之也；不可以来者，无所受之也。物有不通者，故不事也。古人有言曰：「口可以食，不可以言。」言者有讳忌也；「众口铄金」，言有曲故也。

% 译文（白话）
口像机关，用来开闭情意；耳目是心的佐助，用来察奸辨邪。所以说心、耳、口三者参互调谐而后能应物，顺着有利的道去动。所以话多而不乱、纵横而不迷、变易而不危，关键在于睹要得理。
没有目力的人，不必与他炫五色；没有耳力的人，不必与他讲五音。不可以往说的人，是对方闭塞，开不了窍；不可以来受教的人，是对方浅薄，接不住话。物情不通，就不必强为。古人说：「口用来吃饭，别轻易用来发言。」因为言多有讳；又说「众口铄金」，因为众议最容易曲而不直。

```

🤔几点思考：
* 「参调而应」可作演讲前自检：心里想的、眼见的、口说的，是否同一条线。
* 「不可以往…不可以来」不是傲慢，而是止损：沟通也要看可教性与可听性。

<br>

```tex
📕 % 原文
人之情，出言则欲听，举事则欲成。是故智者不用其所短，而用愚人之所长；不用其所拙，而用愚人之所工，故不困也。言其有利者，从其所长也；言其有害者，避其所短也。故介虫之捍也，必以坚厚。螫虫之动也，必以毒螫。故禽兽知用其长，而谈者亦知用其用也。

% 译文（白话）
人之常情：说话想被听，办事想有成。所以聪明人不用己短，而借愚者之长；不用己拙，而借愚者之工，于是少困。说到利处，就顺对方所长去发挥；说到害处，就绕开对方所短以免激反。甲虫自卫，靠的是坚壳；毒虫发动，靠的是毒刺。禽兽尚且知用其长，游说者更要知道此刻该用哪一手。

```

🤔几点思考：
* 「用愚人之所长」不是耍心机，而是组团队、配互补时的常识，用在言辞上即：让对方在他擅长的频道上开口。
* 末句「知用其用」：双「用」字，可理解为既知用己之长，也知用言之宜。

<br>

```tex
📕 % 原文
故曰：辞言有五，曰病、曰怨、曰忧、曰怒、曰喜。故曰：病者，感衰气而不神也；怨者，肠绝而无主也；忧者，闭塞而不泄也；怒者，妄动而不治也；喜者，宣散而无要也。此五者，精则用之，利则行之。故与智者言，依于博；与拙者言，依于辩；与辩者言，依于要；与贵者言，依于势；与富者言，依于高；与贫者言，依于利；与贱者言，依于谦；与勇者言，依于敢；与过者言，依于锐。此其术也，而人常反之。是故与智者言，将以此明之；与不智者言，将以此教之，而甚难为也。故言多类，事多变。故终日言，不失其类，故事不乱。终日不变，而不失其主，故智贵不妄。听贵聪，智贵明，辞贵奇。

% 译文（白话）
所以说：辞令有五种气象，叫病、怨、忧、怒、喜。病，是神气衰而提不起精神；怨，是幽怨断肠而失却主脑；忧，是郁塞不泄；怒，是轻动无条理；喜，是散而无要。这五种，精纯可用、于我有利才推行，否则不如沉默。
于是有「九依」：对智者依博，对拙者依辩，对辩者依要，对贵者依势，对富者依高，对贫者依利，对贱者依谦，对勇者依敢，对有过者依锐。这是常法，世人却常反着用，所以对聪明人要靠这套把理说明，对不聪明的要靠这套去教——后者更难。言多类、事多变，所以即便终日谈说，只要不丢类要，事体就不乱；宗旨终日不变而不失其主脑，智就不妄。收束三句：听要聪，智要明，辞要奇——奇不是猎奇，是切中、出新、能动人。

```

🤔几点思考：
* 「九依」是张弛表：对富谈高（格局与品味），对贫谈利（生计与实惠），不是虚伪，是语境切换。
* 「智贵不妄」可作写作与演讲的戒条：宁可少说，不可妄说。
